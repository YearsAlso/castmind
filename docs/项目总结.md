# 🎯 CastMind 项目总结

## 📋 项目概述

**CastMind** 是一个智能播客订阅处理平台，集成了前端管理界面和后端服务。

## 🏗️ 技术架构

### 后端 (FastAPI)
- **框架**: FastAPI + SQLAlchemy + Pydantic
- **数据库**: SQLite (开发) / PostgreSQL (生产)
- **功能模块**:
  - 订阅源管理 (RSS/Atom 解析)
  - 文章内容抓取和存储
  - AI 智能分析服务
  - 定时任务调度
  - RESTful API 接口

### 前端 (React + TypeScript)
- **框架**: React 18 + TypeScript + Vite
- **样式**: Tailwind CSS
- **状态管理**: TanStack Query
- **路由**: React Router
- **功能页面**:
  - 仪表板 (系统概览和统计)
  - 订阅源管理 (CRUD 操作)
  - 文章管理 (搜索和筛选)
  - 系统管理 (监控和配置)

## 🚀 快速开始

### 1. 安装依赖
```bash
./install.sh
```

### 2. 配置环境变量
```bash
cp .env.example .env
# 编辑 .env 文件配置 API 密钥
```

### 3. 启动服务
```bash
# 启动后端
python backend/main.py

# 启动前端 (另一个终端)
cd frontend
npm install
npm run dev
```

### 4. 访问服务
- **后端 API**: http://localhost:8000
- **API 文档**: http://localhost:8000/api/docs
- **前端界面**: http://localhost:3000
